ON MY TURNTABLE — thumbnails
============================

The "On my turntable" section is a 5x5 grid of 25 covers. The list lives in the
TURNTABLE array near the top of ../../main.js — edit title / artist / url there.

Artwork
- Drop a SQUARE thumbnail here named to match the array: 01.webp ... 25.webp
- Keep them small: ~300px square WebP (20-40 KB each) is plenty for the grid.
- Any tile whose image is missing falls back to a coloured gradient with the
  title, so the grid never looks broken while you're filling it in.

Listening
- Each tile is a link that opens `url` in a new tab — point it at wherever you
  want people to listen (Spotify / Bandcamp / Apple / YouTube / SoundCloud).
