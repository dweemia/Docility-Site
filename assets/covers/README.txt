ALBUM COVER ARTWORK
===================

These four album covers are already in place (copied from your Drive:
F:\My Drive\Music Projects\Production\...). To update one, just replace the file,
keeping the same filename:

    prism.png         → "Prism"
    reflections.png   → "Reflections"
    open-sky.png      → "Open Sky"
    contact.png       → "Contact EP"

Notes
- They should be square. If you swap in a different format (JPG/WebP), update the
  matching `cover:` line in ../../main.js so the extension agrees.
- If a file is missing, that tile falls back to a coloured gradient with the album
  title, so the site never looks broken.
